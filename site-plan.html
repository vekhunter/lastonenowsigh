Raw V Gaming Rig - Site Plan

Purpose and Audience

Site Purpose: The purpose of this site is to showcase and sell a custom PC rig designed with a focus on stability, airflow, and clean power management.

Intended Audience: PC buyers who want a reliable, cool-running system and people who want to learn how to build stable rigs without cutting corners.

Dynamic Elements (JavaScript)

JavaScript will be used to enhance user interactivity and provide a more engaging experience:

  • Collapsible Sections: Clicking on section headings will toggle the content open and closed. An animated arrow icon rotates to indicate the current state. This keeps the page organized and allows users to focus on specific information.
  • Interactive Navigation: Smooth scrolling and active link highlighting help users understand their location on the site.
  • Image Gallery: On the component and performance pages, JavaScript can enable lightbox functionality to view full-resolution images without leaving the page.

These dynamic elements make the site feel more responsive and professional while maintaining fast load times and accessibility.
