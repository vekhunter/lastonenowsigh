  • GPU: We use GTX 970s—a great esports GPU. Our goal with this GPU and the setup is not to create a high settings rig but to create a smooth and cheap rig. The 970 reflects that, and it's stylish too. Although, if you want to play really demanding games, we recommend an RTX 3060.

    Team Red Note: If you're on Team Red (AMD), get 8GB of VRAM with the RX 580 for the best experience.
  • Motherboard: ASRock H110 Pro BTC+ — this component is originally designed for mining, but since mining has died there is no demand for it. It's not bad at all: it has a stable power setup and is designed to run 24/7, which is more than you will ever need.
  • Power Supply: Corsair RMx 850W — one of the most reliable power supplies ever made. We never compromise on the PSU, which ensures stability and a stable system.
  • CPU Cooler: Hyper 212 + MX-4 — the Hyper 212 is a classic reliable cooler and is crazy cost effective for the rig. The same goes for Arctic MX-4 thermal paste: the thermal paste is high quality, can last for a year or longer, and helps keep our temps low.
  • Frame: Mining frame — we use high quality mining frames that are open air, which helps keep temps low instead of being cramped in a box. This also makes it easier for you to build and trade out components, and if you ever need, install up to 6 GPUs.
  • Storage: We use an M.2 for the main OS and use SATA 2.5 inch SSDs for game storage, keeping it fast where it needs it but saving you money. We like using Crucial and Intel, ensuring reliability.
  • Fans: We use Arctic F12s for their incredible cost but also for their reliability. They last a very long time and are really good quality—we love them from Arctic. They draw very little power but push a lot of air.
